Subsets and the Empty Set
For two sets $A$ and $B$, if every element of $A$ is also an element of $B$, then $A$ is called a subset of $B$, written as $A \subset B$.
Also, a set with no element is called the empty set and is written as $ \varnothing $.
💡
The empty set $ \varnothing $ is considered a subset of every set.
