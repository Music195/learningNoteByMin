Average (Mean) of Data

Data Analysis
For a variable \( x \), if the data values are \( x_1, x_2, \ldots, x_n \), the average (mean) is the sum of all values divided by \( n \), and is denoted as \( \overline{x} \):
\( \overline{x} = \frac{1}{n} \left( x_1 + x_2 + \cdots + x_n \right) \)
💡 The characteristics of the entire data set can often be represented by a single number, called a representative value.
Common representative values are: mean (average), median, mode.
