Properties of Exponential Functions
About \( y = a^x \):
1. The domain is all real numbers, and the range is all positive real numbers.

2. If \( a > 1 \), the function is increasing:
 If \( p < q \), then \( a^p < a^q \).

3. If \( 0 < a < 1 \), the function is decreasing:
 If \( p < q \), then \( a^p > a^q \).
When \( a > 1 \): Increasing function
When \( 0 < a < 1 \): Decreasing function
When the value of x increases and y also increases, the function is called an increasing function.
When the value of x increases and y decreases, the function is called a decreasing function.
