Properties of Logarithmic Functions
About \( y = \log_a x \):
1. The domain is all positive real numbers, and the range is all real numbers.

2. If \( a > 1 \), the function is increasing:
 If \( 0 < p < q \), then \( \log_a p < \log_a q \).

3. If \( 0 < a < 1 \), the function is decreasing:
 If \( 0 < p < q \), then \( \log_a p > \log_a q \).
When \( a > 1 \): Increasing function
When \( 0 < a < 1 \): Decreasing function
