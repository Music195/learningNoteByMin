Distance Between a Point and a Line
The distance \( d \) between a point \( (x_1, y_1) \) and a line \( ax + by + c = 0 \) is given by:

\( d = \frac{|ax_1 + by_1 + c|}{\sqrt{a^2 + b^2}} \)

Specifically, the distance from the origin \( O \) to the line \( ax + by + c = 0 \) is:

\( d = \frac{|c|}{\sqrt{a^2 + b^2}} \)
The formula uses the absolute value to ensure the distance is always positive.
