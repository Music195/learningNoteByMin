Set and Elements

Introduction to Sets in Mathematics
If x is an element of set A, we write x ∈ A.
If y is not an element of set A, we write y ∉ A.
The set B of all positive divisors of 15 is:
B = { 1, 3, 5, 15 } or B = { x | x is a positive divisor of 15 }
Note: The notation { x | condition } means "the set of all x such that the condition holds."
