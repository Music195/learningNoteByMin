Basic Properties of Probability

Fundamental Rules for Probability
  • For any event A: 0 ≤ P(A) ≤ 1
  • For the empty set ∅: P(∅) = 0
    For the universal set U: P(U) = 1
  • If events A and B are mutually exclusive:
    P(A ∪ B) = P(A) + P(B)
The third property is called the addition law of probability.
For more than two mutually exclusive events, the same rule applies.
